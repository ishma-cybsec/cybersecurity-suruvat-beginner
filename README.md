This repository contains a set of small Python utilities developed to practice foundational cybersecurity concepts, including password hygiene, data confidentiality, and multi-factor authentication.

The tools are intentionally simple and terminal-based, focusing on clarity, correctness, and ethical use rather than advanced exploitation techniques.


---

Included Tools

1. Password Strength Checker

A command-line utility that evaluates passwords based on basic security criteria such as length and character variety.
Learning focus:
Password hygiene
Security policy awareness
Input handling and validation



---

2. Encrypt / Decrypt Utility

A simple encryption and decryption tool demonstrating how data confidentiality can be implemented using standard cryptographic libraries.

Learning focus:
Symmetric encryption concepts
Secure handling of sensitive data
Correct use of cryptographic primitives



---

3. One-Time Password (OTP) Generator

A basic two-factor authentication utility using time-based one-time passwords (TOTP).

Learning focus:

Multi-factor authentication principles
Time-based security mechanisms
Authentication workflow basics



---

Technical Requirements

Language: Python 3.8+

Libraries:

cryptography
pyotp



Install dependencies using:

pip install cryptography pyotp


---

Ethical & Educational Scope

These tools are intended strictly for educational purposes and controlled learning environments.
They do not perform unauthorized access, brute-force attacks, or invasive testing.

The emphasis is on understanding defensive security principles and responsible software design.


---

Learning Outcomes

By working with this repository, the following skills are demonstrated:
Understanding of core cybersecurity fundamentals
Practical application of encryption and authentication concepts
Writing clear, maintainable, and auditable Python code
Ethical awareness in security-related development
